# Цветовая палитра LUA — Инструкция для сайта

## Обзор палитры

Палитра LUA состоит из трёх цветовых групп: **тёплые тона**, **холодные тона** и **акцентные тона**. Каждая группа содержит градацию от светлого к тёмному для создания глубины и иерархии.

---

## Тёплые тона (Gold / Brown)

Используются для создания уюта, тепла и премиальности.

| Название | HEX | RGB | Применение |
|----------|-----|-----|------------|
| **Cream** | `#EFEEE9` | 239, 238, 233 | Фон страниц, светлые секции |
| **Sand** | `#E2CA8C` | 226, 202, 140 | Акцентные блоки, hover-эффекты |
| **Gold** | `#E1A81B` | 225, 168, 27 | CTA-кнопки, важные акценты |
| **Bronze** | `#935F10` | 147, 95, 16 | Подзаголовки, иконки |
| **Brown** | `#593C0F` | 89, 60, 15 | Тёмный текст на светлом фоне |

---

## Холодные тона (Blues)

Основная палитра для текста и структуры. Тёмно-синий заменяет чёрный.

| Название | HEX | RGB | Применение |
|----------|-----|-----|------------|
| **Ice Blue** | `#C9DCED` | 201, 220, 237 | Светлый фон, разделители |
| **Sky Blue** | `#8EA8CB` | 142, 168, 203 | Вторичные элементы, ссылки hover |
| **Steel Blue** | `#536C95` | 83, 108, 149 | Иконки, подписи |
| **Navy** | `#253449` | 37, 52, 73 | Основной текст (вместо чёрного) |
| **Deep Navy** | `#04172F` | 4, 23, 47 | Заголовки, тёмные акценты |

---

## Акцентные тона (Pink / Burgundy)

Бордовый — акцентный цвет для выделения и создания контраста.

| Название | HEX | RGB | Применение |
|----------|-----|-----|------------|
| **Pearl** | `#F8F3F0` | 248, 243, 240 | Альтернативный светлый фон |
| **Blush** | `#FAE8E9` | 250, 232, 233 | Нежные выделения, карточки |
| **Burgundy** | `#921C38` | 146, 28, 56 | Основной акцент, CTA |
| **Wine** | `#7B1830` | 123, 24, 48 | Hover для бордового, ссылки |
| **Dark Wine** | `#551624` | 85, 22, 36 | Тёмный текст на светлом акценте |

---

## Правила использования

### ✅ Для текста
- **Основной текст:** `#253449` (Navy) — не используем чистый чёрный
- **Заголовки:** `#04172F` (Deep Navy) или `#551624` (Dark Wine)
- **Светлый текст на тёмном фоне:** `#FFFFFF` или `#F8F3F0` (Pearl)

### ✅ Для фонов
- **Основной фон:** `#FFFFFF` или `#EFEEE9` (Cream)
- **Альтернативные секции:** `#F8F3F0` (Pearl) или `#C9DCED` (Ice Blue)
- **Тёмные секции:** `#04172F` (Deep Navy) или `#253449` (Navy)

### ✅ Для акцентов
- **Кнопки CTA:** `#921C38` (Burgundy) или `#E1A81B` (Gold)
- **Hover-состояния:** переход к более тёмному оттенку в той же группе
- **Ссылки:** `#536C95` (Steel Blue), hover: `#253449` (Navy)

### ❌ Запрещено
- Чистый чёрный `#000000` для текста
- Смешивание более 3-4 цветов на одном экране
- Бордовый и золотой рядом без нейтрального разделителя

## CSS-переменные

```css
:root {
  /* Warm Tones */
  --color-cream: #EFEEE9;
  --color-sand: #E2CA8C;
  --color-gold: #E1A81B;
  --color-bronze: #935F10;
  --color-brown: #593C0F;
  
  /* Cool Tones */
  --color-ice-blue: #C9DCED;
  --color-sky-blue: #8EA8CB;
  --color-steel-blue: #536C95;
  --color-navy: #253449;
  --color-deep-navy: #04172F;
  
  /* Accent Tones */
  --color-pearl: #F8F3F0;
  --color-blush: #FAE8E9;
  --color-burgundy: #921C38;
  --color-wine: #7B1830;
  --color-dark-wine: #551624;
  
  /* Semantic Variables */
  --color-text-primary: var(--color-navy);
  --color-text-heading: var(--color-deep-navy);
  --color-text-light: var(--color-pearl);
  --color-background: #FFFFFF;
  --color-background-alt: var(--color-cream);
  --color-accent-primary: var(--color-burgundy);
  --color-accent-secondary: var(--color-gold);
  --color-link: var(--color-steel-blue);
  --color-link-hover: var(--color-navy);
}
```

## Рекомендуемые сочетания

### Светлая тема (основная)
```
Фон: #FFFFFF или #EFEEE9
Текст: #253449
Заголовки: #04172F
Акценты: #921C38
```

### Тёмная секция
```
Фон: #04172F
Текст: #FFFFFF
Заголовки: #E2CA8C
Акценты: #E1A81B
```

### Нежная/свадебная секция
```
Фон: #F8F3F0
Текст: #551624
Заголовки: #7B1830
Акценты: #921C38
```

| Сочетание | Контраст | Уровень |
|-----------|----------|---------|
| Navy на White | 12.5:1 | AAA ✓ |
| Deep Navy на Cream | 13.8:1 | AAA ✓ |
| Burgundy на White | 8.2:1 | AAA ✓ |
| White на Deep Navy | 15.3:1 | AAA ✓ |
| Gold на Deep Navy | 8.9:1 | AAA ✓ |
