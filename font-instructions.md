# Инструкция по использованию шрифтов для сайта LUA

## Шрифты для сайта

На сайте используются **три шрифта**:

| Шрифт | Роль | Характеристика |
|-------|------|----------------|
| **Prata** | Заголовки, акценты | Антиква, элегантный, поддерживает латиницу и кириллицу |
| **Montserrat** | Основной текст, подписи | Гротеск, нейтральный, поддерживает латиницу и кириллицу |
| **Libre Franklin** | Наборный текст | Дополнительный шрифт для длинных текстовых блоков |

---

## Применение шрифтов

### Prata
- **Где:** главные заголовки страниц, названия разделов, крупные акцентные надписи
- **Почему:** формирует визуальную идентичность бренда, вдохновлённую эстетикой Vogue Weddings
- **Важно:** это «лицо» бренда — использовать последовательно во всех ключевых заголовках

### Montserrat
- **Где:** подзаголовки, навигация, кнопки, небольшие подписи, структурные блоки
- **Почему:** нейтральный и аккуратный, не конкурирует с Prata, сохраняет воздушность

### Libre Franklin
- **Где:** длинные текстовые блоки (описания, статьи, информационные разделы)
- **Почему:** хорошо читается в наборе, дополняет основную пару шрифтов

---

## Допустимые комбинации

✅ **Prata** (крупное) + **Montserrat** (мелкое)  
✅ **Prata** + **Libre Franklin** (для наборного текста)  
✅ **Montserrat** + **Libre Franklin**

❌ Не рекомендуется использовать все три шрифта на одной странице/секции — это усложняет визуал и нарушает гармонию

## Цвета текста

### Основные правила
- **Не используем чистый чёрный (#000000)** для шрифтов
- Вместо чёрного применяем:
  - Тёмно-синий
  - Тёмно-коричневый
  - Тёмно-бордовый

### Для светлого фона
Используем тёмные оттенки из палитры бренда

### Для тёмного фона
- Белый — базовый цвет текста
- Обеспечивает максимальную читаемость

### Подложки
Если текст плохо читается на фоне изображения — используем подложку:
- Чёрная заливка с мягким градиентом
- Прозрачность 50–60%

## Грузинский текст

При необходимости использования грузинского языка:
- **Шрифт:** BPG Nino Mtavruli
- **В паре с:** Montserrat (для английского текста)
- **Рекомендация:** по возможности избегать смешения грузинского и английского шрифтов на одном экране

## Иерархия типографики (рекомендация для CSS)

```
h1, h2         → Prata
h3, h4, h5, h6 → Montserrat или Prata (в зависимости от контекста)
body, p        → Libre Franklin или Montserrat
nav, button    → Montserrat
caption, small → Montserrat
```

## Подключение шрифтов

### Google Fonts
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Libre+Franklin:wght@400;500;600&family=Montserrat:wght@400;500;600;700&family=Prata&display=swap" rel="stylesheet">
```

### CSS переменные
```css
:root {
  --font-heading: 'Prata', serif;
  --font-body: 'Libre Franklin', sans-serif;
  --font-ui: 'Montserrat', sans-serif;
}
```
